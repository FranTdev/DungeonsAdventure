# DungeonsAdventure V0.1
## About The Project

DungeonsAdventure is a console-based text RPG engine written in C#. This project serves as a demonstration of fundamental Object-Oriented Programming (OOP) and clean architecture principles. The current version (v1.0) features a robust interaction system where a player can interact with various objects within a single room.

---

## Getting Started

To run this project, you need the .NET SDK installed.

1.  Clone the repository:
    `(https://github.com/FranTdev/DungeonsAdventure.git)`
2.  Navigate to the project directory:
    `cd DungeonsAdventure`
3.  Run the application:
    `dotnet run`
